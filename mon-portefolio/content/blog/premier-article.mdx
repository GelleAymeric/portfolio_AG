## Bienvenue sur le blog !

Ceci est un exemple d'article de blog écrit en **MDX**. Le MDX vous permet d'utiliser la syntaxe Markdown que vous connaissez et aimez, tout en vous donnant la possibilité d'intégrer des composants React directement dans votre contenu.

### Fonctionnalités intéressantes

-   **Markdown :** Écrivez en Markdown, c'est simple et rapide.
-   **Composants React :** Importez et utilisez des composants React pour du contenu interactif.
-   **Coloration syntaxique :** Les blocs de code sont automatiquement mis en évidence.

Voici un exemple de bloc de code :

```tsx
import React from 'react';

const MonComposant = () => {
  return <div>Salut le monde !</div>;
}

export default MonComposant;
```

Vous pouvez même importer un composant personnalisé ici si vous le souhaitez. Nous verrons cela plus tard !
